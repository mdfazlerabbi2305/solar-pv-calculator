# Solar PV System Calculator

A comprehensive, web‑based tool for sizing and checking key components of a photovoltaic (PV) system, following international standards (IEC, NEC, IEEE).

## Features
- **Solar Panel** – required peak power based on daily load, sun hours, and performance ratio.
- **Battery** – capacity (Ah) and backup time (hours) with autonomy, DoD, SoH, efficiency.
- **Inverter** – sizing with safety factor.
- **Charge Controller** – current rating for PWM / MPPT.
- **MCB** – overcurrent protection for DC and AC circuits.
- **SPD** – surge protection device voltage/current ratings.
- **Wiring** – minimum cable cross‑section with voltage drop.
- **Structure** – basic geometry for tilted panels.

All calculations are referenced to IEC 61215, IEC 61427, IEEE 1547, NEC 690, and other relevant standards.

## How to Use
1. Open the [live demo](https://mdfazlerabbi2305.github.io/solar-pv-calculator/) or clone the repository.
2. Navigate through the tabs to access different calculators.
3. Enter the required values and click **Calculate**.
4. Results are displayed instantly with clear units.
